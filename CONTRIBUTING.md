## ?? Getting Started

### 1. Fork the Repository

Click the **Fork** button at the top right of the repo.

This makes your own copy you can edit.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/wgnd1017.git
cd wgnd1017
```

### 3. Make Your Changes

You can use a simple editor like [VS Code](https://code.visualstudio.com/) or even GitHub’s web editor.

### 4. Preview Your Changes (optional)

Open `index.html` in your browser locally to see your edits.

### 5. Submit a Pull Request

When ready:

1. Go to your fork on GitHub
2. Click **Pull Request**
3. Describe your changes
4. Submit!

---

## ?? Tips for Beginners

- Make small changes at a time
- Ask questions — we’re here to help!
- Use clear commit messages (e.g., `Add July sermon`)
